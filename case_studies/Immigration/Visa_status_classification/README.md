IMMIGRATION

Visa Status Classification

Description

Analyze the data of visa applicants and build a predictive model to streamline the visa approval process. 
The model will identify important factors influencing visa status and recommend suitable profiles for applicants,
determining whether their visa should be certified or denied.


It Includes:
    Exploratory Data Analysis
    Data Preprocessing
    Bagging
    Random Forest
    Boosting
    AdaBoost
    Gradient Boosting
    XGBoost
    GridSearchCV
